# 🏆 Estudos de Programação Competitiva

Este repositório contém meus estudos, práticas e resoluções de problemas voltados para a disciplina de **Introdução aos Conceitos de Programação Competitiva** da UFG.

O projeto está estruturado em duas partes principais, refletindo diferentes etapas do meu aprendizado:

## 📂 Estrutura do Repositório

### 1. [Estudos_ICPC](./Estudos_ICPC/)
Esta pasta é dedicada ao **estudo preparatório** para a matéria. Ela contém as resoluções de listas de exercícios e atividades do período de **2025.1**. O objetivo aqui foi construir a base lógica e algorítmica necessária antes de iniciar os tópicos oficiais da disciplina.
- **Conteúdo:** Treinos práticos (Treino 1 a Treino 6).

### 2. [UFG_ICPC_2026.1](./UFG_ICPC_2026.1/) *(Em Progresso 🚀)*
Esta pasta refere-se à **matéria de Introdução aos Conceitos de Programação Competitiva propriamente dita**, em andamento no semestre **2026.1**. Aqui documento os exercícios oficiais da disciplina, aprofundando o conhecimento em algoritmos avançados, estruturas de dados otimizadas e técnicas clássicas de maratonas de programação.
- **Conteúdo:** Treinos e listas da matéria (Treino 0, Treino 1, etc.).

---

## 🎯 Objetivos do Projeto
- Consolidar conhecimentos em Algoritmos e Estruturas de Dados.
- Aprimorar o raciocínio lógico focado em resolução rápida de problemas.
- Estudar tópicos frequentes em maratonas de programação (ICPC, Maratona SBC, etc.), tais como Grafos, Programação Dinâmica, Algoritmos Gulosos, Matemática e Geometria Computacional.
- Melhorar a capacidade de analisar otimização de tempo de execução e uso de memória.

## 🛠️ Tecnologias e Linguagens Utilizadas
- **C++**: Linguagem exclusiva utilizada para a resolução de todos os exercícios de programação competitiva presentes neste repositório.
